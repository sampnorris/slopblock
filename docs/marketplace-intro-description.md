# Introductory Description

<!-- The opening paragraph of the Marketplace listing. Sets context before the feature list. -->

SlopBlock gates pull request merges behind a short, AI-generated comprehension quiz grounded in the actual diff. When a PR is opened, SlopBlock reads the changed code, generates multiple-choice questions about what the code does and what could go wrong, validates them for accuracy with a second model, and posts a link to the quiz directly in the PR. The merge stays blocked until the author proves they understand the changes. Trivial PRs — docs, tests, lockfile bumps, formatting — are automatically skipped so the quiz only appears when it matters. Designed for teams shipping AI-assisted code who want to make sure every merge is backed by real comprehension, not just a rubber stamp.
